# 💪 Personalized Fitness Plan Generator

---

## 🌟 Overview

This milestone focuses on building an intelligent system that analyzes user health data and generates **personalized fitness plans** using BMI (Body Mass Index). It integrates user input, validation, logical processing, and deployment into a functional web application.

---

## 🎯 Objective

The goal of this milestone is to:

✔️ Collect user health details through an interactive form.

✔️ Validate user inputs for accuracy and reliability.

✔️ Calculate BMI using a standard formula.

✔️ Categorize users into fitness levels.

✔️ Generate customized fitness plans based on BMI.

---

## ⚖️ BMI Formula

[
\textbf{BMI} = \frac{\text{Weight (kg)}}{\text{Height (m)}^2}
]

### 📊 BMI Classification

| Category       | BMI Range   | Fitness Suggestion 💡  |
| -------------- | ----------- | ---------------------- |
| 🟦 Underweight | < 18.5      | Weight gain plan       |
| 🟩 Normal      | 18.5 – 24.9 | Maintain fitness       |
| 🟨 Overweight  | 25 – 29.9   | Fat loss plan          |
| 🟥 Obese       | ≥ 30        | Intensive fitness plan |

## 🛠️ Implementation Steps

### 🧾 1. Form Creation

* Designed a clean and responsive user interface
* Captured user inputs:

  * Age 👤
  * Height 📏
  * Weight ⚖️
  * Fitness Goal 🎯

---

### ✅ 2. Input Validation

* Ensured all required fields are filled
* Applied validation rules:

  * Only numeric values allowed
  * No negative or zero values
* Displayed real-time error messages ⚠️

---

### 🧠 3. BMI Calculation Logic

* Converted height into meters (if entered in cm).

* Applied BMI formula.

* Categorized the result into BMI ranges.

* Mapped each category to a suitable fitness plan.

---

### 🌐 4. Deployment

* Hosted the project on a web platform 🌍
* Tested for:

  * Responsiveness 📱
  * Accuracy ✔️
  * Performance ⚡
* Ensured smooth user experience
